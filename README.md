# 🗺️ Map Route Finder using Dijkstra’s Algorithm

## 📌 Project Description

This project finds the shortest path between cities using Dijkstra’s Algorithm and displays it visually on a map.

## 🚀 Features

* Shortest path calculation
* Map-based visualization
* User-friendly GUI (Tkinter)
* Distance display
* Input validation

## 🛠️ Technologies Used

* Python
* Tkinter
* PIL (Image Processing)
* Dijkstra Algorithm

## ▶️ How to Run

1. Install dependencies:

   ```
   pip install pillow
   ```
2. Run the file:

   ```
   python MAIN.py
   ```

## 📷 Output

* Displays shortest route on map
* Shows distance in km
